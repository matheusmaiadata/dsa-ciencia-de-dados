# Projeto 7 - IA Para Previsão de Demanda e Otimização do Estoque de Produtos Sazonais

# Instruções:

# 1- Instale o Docker Desktop conforme instruções das aulas.

# 2- Abra o terminal ou prompt de comando e execute a instrução abaixo para criar o container Docker:

docker run --name dsa-projeto7 -p 5551:5432 -e POSTGRES_USER=dsa -e POSTGRES_PASSWORD=dsa1010 -e POSTGRES_DB=dsadb -d postgres:16.1

# 4- Abra o terminal ou prompt de comando, navegue até a pasta com os arquivos e execute o comando abaixo para criar um ambiente virtual:

uv init

# 6- Abra o terminal ou prompt de comando, navegue até a pasta onde estão os arquivos e instale uv e as dependências: 

uv add -r requirements.txt

# 7- Abra o terminal ou prompt de comando, navegue até a pasta onde estão os arquivos e execute o pipeline assim:

python Projeto7-05-Run-Pipeline.py

# Use os comandos abaixo para desativar o ambiente virtual e remover o ambiente (opcional):

conda deactivate
conda remove --name dsabusinessp7 --all
